# frontend\src\features\plugins

## Responsibility

Plugin discovery, capability approval, and plugin management UI.

## Belongs here

- Feature-specific React components and hooks
- Local UI state
- Calls to the typed API client

## Does not belong here

- Direct filesystem or process control
- Domain policy implementation
- SQLite access
