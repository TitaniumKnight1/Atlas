# backend\domain\config

## Responsibility

Domain models, policies, events, and port interfaces for Configuration editing policies, validation rules, undo history, and diff semantics.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
