# backend\domain\automation

## Responsibility

Domain models, policies, events, and port interfaces for Visual workflow definitions, triggers, conditions, actions, and approval policies.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
