# backend\domain\monitoring

## Responsibility

Domain models, policies, events, and port interfaces for CPU, memory, disk, FPS, players, network, database, and resource health metrics.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
