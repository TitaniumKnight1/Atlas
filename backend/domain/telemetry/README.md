# backend\domain\telemetry

## Responsibility

Domain models, policies, events, and port interfaces for Atlas application telemetry policy, sanitization rules, and user opt-out controls.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
