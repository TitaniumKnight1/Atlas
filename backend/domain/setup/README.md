# backend\domain\setup

## Responsibility

Domain models, policies, events, and port interfaces for Artifact selection, FXServer setup, txAdmin guidance, and initial server configuration.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
