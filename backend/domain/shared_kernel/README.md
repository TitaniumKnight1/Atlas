# backend\domain\shared_kernel

## Responsibility

Minimal cross-module primitives shared by all bounded contexts.

## Belongs here

- Project and environment identifiers
- Severity levels
- Domain event envelope
- Audit metadata
- Result and error primitives

## Does not belong here

- Business rules for specific modules
- Adapter-specific types
- ORM models
