# backend\domain\backup

## Responsibility

Domain models, policies, events, and port interfaces for Backup scheduling, compression, retention, restore, and snapshot catalog.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
