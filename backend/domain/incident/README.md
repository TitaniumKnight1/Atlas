# backend\domain\incident

## Responsibility

Domain models, policies, events, and port interfaces for Local incident capture, fingerprinting, deduplication, timeline, and Markdown export.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
