# backend\domain\resources

## Responsibility

Domain models, policies, events, and port interfaces for Resource install, update, enable, disable, delete, rollback, and dependency graph.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
