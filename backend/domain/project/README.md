# backend\domain\project

## Responsibility

Domain models, policies, events, and port interfaces for Project import, workspace management, environment profiles, templates, and trust state.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
