# backend\domain\plugin

## Responsibility

Domain models, policies, events, and port interfaces for Plugin discovery, manifest validation, capability approval, and contribution registry.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
