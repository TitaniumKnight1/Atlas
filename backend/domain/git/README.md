# backend\domain\git

## Responsibility

Domain models, policies, events, and port interfaces for Git clone, branch, commit, pull, diff, and local modification detection.

## Belongs here

- Entities and value objects
- Domain policies
- Port interfaces
- Domain events

## Does not belong here

- Application orchestration
- Adapter implementations
- API schemas
