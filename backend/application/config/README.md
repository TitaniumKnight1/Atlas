# backend\application\config

## Responsibility

Application services orchestrating commands and queries for Configuration editing policies, validation rules, undo history, and diff semantics.

## Belongs here

- Command handlers
- Query handlers
- Use-case orchestration

## Does not belong here

- FastAPI routers
- Concrete adapter code
- ORM mapping details
