# backend\application\telemetry

## Responsibility

Application services orchestrating commands and queries for Atlas application telemetry policy, sanitization rules, and user opt-out controls.

## Belongs here

- Command handlers
- Query handlers
- Use-case orchestration

## Does not belong here

- FastAPI routers
- Concrete adapter code
- ORM mapping details
