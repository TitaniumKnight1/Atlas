# backend\application\resources

## Responsibility

Application services orchestrating commands and queries for Resource install, update, enable, disable, delete, rollback, and dependency graph.

## Belongs here

- Command handlers
- Query handlers
- Use-case orchestration

## Does not belong here

- FastAPI routers
- Concrete adapter code
- ORM mapping details
