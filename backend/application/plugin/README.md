# backend\application\plugin

## Responsibility

Application services orchestrating commands and queries for Plugin discovery, manifest validation, capability approval, and contribution registry.

## Belongs here

- Command handlers
- Query handlers
- Use-case orchestration

## Does not belong here

- FastAPI routers
- Concrete adapter code
- ORM mapping details
